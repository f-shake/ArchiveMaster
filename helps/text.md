# 文本智能生成

文本智能生成工具是一款基于AI大模型的多功能文本处理助手，能够根据用户需求对文本进行智能生成、润色、改写、结构调整、内容变换与翻译等操作。它不仅适用于写作创作、学术论文润色、内容改写与优化场景，也可用于文案生成、语言表达规范化及多语言翻译，为用户提供高效、灵活、个性化的文本生成方案。

## 功能特性

1. 多类型文本生成：支持表达优化、结构调整、内容变换、文本评价、自定义提示等多种模式，满足从写作到校对的多样化需求。
2. 智能语义优化：可针对语句流畅度、逻辑性与表达清晰度进行AI优化，生成自然流畅的文本内容。
3. 多语言翻译支持：支持将文本自动翻译为中文、英文、日文、韩文等语言，保持语义一致与表达自然。
4. 灵活自定义提示词：用户可自由输入自定义指令，定制AI输出风格与内容方向，满足专业与创意需求。
5. 实时生成反馈：采用流式AI输出机制，实时显示生成内容，提升交互体验与处理效率。
6. 强大兼容性：支持多种文本来源（如文件、剪贴板、直接输入等），适配不同使用场景。

## 配置说明

1. 文本源：选择或输入需要处理的文本内容来源，可为文件、剪贴板或直接输入文本。
2. 类型选择：在“表达优化”“结构调整”“内容变换”“文本评价”或“自定义”中选择AI处理类型。
3. 模式选项：不同类型对应不同的细分模式，例如：
   - 表达优化：可选择“正式化”“简化”“润色”等子类型。
   - 结构调整：可选择“扩展语句”“压缩内容”“句式变换”等子类型。
   - 内容变换：包括“翻译”“改写”等功能。
   - 文本评价：可进行内容逻辑、语言质量等维度的AI评估。
   - 自定义：手动指定提示词
4. 翻译目标语言：当选择“翻译”模式时，指定目标语言（中文、英文、日文、韩文等）。
5. 自定义提示词：在自定义模式下，填写完整AI提示内容，以完全控制生成风格与任务。
6. 额外提示：为非自定义模式提供补充说明，用于调整或强化AI输出效果。

## 原理解析

该工具通过读取用户提供的文本内容，根据所选的处理类型自动生成相应的AI提示词，调用智能语言模型对文本进行理解与再创作。系统在生成过程中实时接收AI输出结果，并动态展示给用户，实现从文本输入到优化输出的全流程智能化处理。工具内部采用分步执行与流式反馈机制，确保生成结果既符合用户设定的风格与目标，又保持语言表达的自然与连贯。

# 文本混淆

文本混淆工具是一款用于对文本内容进行加密与解密处理的安全辅助工具，旨在保护敏感信息或隐藏关键信息片段。它支持对选定范围或整段文本进行灵活混淆，可通过自定义前后缀标记来区分加密区域，从而实现明文与密文的共存与自动识别，适用于保密文档、配置文件或敏感数据的安全传输场景。

## 功能特性

1. 双向加密与解密：支持将文本内容进行加密或解密，自动识别加密标记区域，实现部分内容保护。
2. 前后缀标记机制：通过自定义前缀与后缀（如“##”），灵活划定需加密的文本范围，未标记部分保持明文。
3. 安全密码保护：采用安全密码输入机制，防止明文密码暴露，确保加密过程的安全性与一致性。
4. 自动任务执行：当输入文本或配置参数变化时，系统可自动触发上一次操作（加密或解密），无需手动重复执行。
5. 任务队列控制：引入任务队列机制，确保频繁变更时仅执行最后一次有效操作，避免资源竞争与重复计算。
6. 大文本处理优化：支持最大约30万字符的文本处理能力，并对超限文本进行安全检测与提示。
7. 可视化操作界面：提供直观的图形化界面，支持输入源选择、密码设定、前后缀配置及实时结果展示。

## 配置说明

1. 文本源：可选择文件输入或直接输入文本。
2. 加密密码：设置用于加密与解密的密码。
3. 密文前缀与后缀：用于标识加密区域的符号，需成对出现。
4. 操作类型：支持“加密”与“解密”两种操作，可手动执行或自动触发。
5. 自动模式：当启用时，系统会在文本或配置变更后自动重复上次操作。
6. 任务状态：界面提供实时执行状态显示，操作完成后可直接查看结果文本。

## 原理解析

该工具的核心机制是基于“可逆映射”的文本置换算法，目标是在不依赖外部加密库的前提下，实现对文本中指定内容的安全混淆与还原。其实现思路源于“字符分区 + 确定性乱序”的原理，即通过对字符进行分组、置换和映射，确保同一口令下的加密与解密操作严格对应，具有可重复性与可逆性。

首先，系统会扫描输入文本，依据预设的前后缀标记确定需要处理的文本范围。随后，它构建一个“字符区间查找表”，将所有可处理字符按 Unicode 编码划分为若干区段，例如拉丁字母区、汉字区、假名区等，并在分区过程中自动忽略控制字符、空格、标点等无须处理的符号。这样的分区设计使算法能在不同语言环境下稳定工作，同时避免了对特殊字符的错误替换。

在置换阶段，系统使用用户提供的口令生成“区间随机种子”。这一过程通过轻量级的哈希运算，将口令转化为一个可重复的数值序列，并将该序列作为伪随机数发生器的初始状态。随后，算法针对每个字符区间执行 Fisher–Yates 洗牌，生成“乱序表”与对应的“反向表”。乱序表定义了原字符到密文字符的映射关系，而反向表则用于解密时的反向还原。由于洗牌结果由口令唯一决定，因此同一口令在任意时间执行都能得到一致的映射结果。

加密过程中，系统逐个遍历文本字符，判断其所属区间，并依据乱序表完成替换；未在可处理范围内的字符则保持原样。解密过程则完全相反，利用反向表将密文字符映射回原字符。整个过程不改变文本长度，也不破坏其结构特征，因此适合用于代码、配置文件或自然语言文本的轻量级保护。

在性能设计上，工具采用分段流式处理机制。对于大体量文本，会动态拆分为多个逻辑片段并通过任务队列执行，从而避免内存占用过高；同时在执行前增加字符数量上限检测与后台任务状态管理，以防止长时间阻塞。最终，系统在保证处理效率与交互流畅性的基础上，实现了基于口令的可逆字符混淆功能，使得用户既能便捷地隐藏文本信息，又能在需要时精确恢复原文。

# 文档智能搜索

文档智能搜索工具是一款集关键词检索与AI总结于一体的智能文档分析系统，适用于批量文本或文件内容的快速搜索与语义提炼。它支持普通匹配与正则表达式搜索，能够在文档中高效定位目标信息，并自动生成上下文摘要，帮助用户快速理解关键信息内容。

## 功能特性

1. 多源文本搜索：支持从文件或直接输入的文本中进行检索，可灵活切换输入方式。
2. 多关键词匹配：可同时输入多个关键词进行搜索，支持模糊匹配与正则表达式模式。
3. 上下文提取：自动截取每个关键词所在位置的前后文，生成便于理解的语义片段。
4. 智能结果合并：对重叠或相邻的搜索结果进行合并，避免重复内容，提高展示的连贯性。
5. AI归纳总结：在搜索完成后，可调用语言模型对所有结果进行归纳分析，自动生成精炼总结。
6. 流式输出反馈：AI总结过程支持实时输出，用户可动态查看模型生成的内容。
7. 参数化控制：可配置搜索上下文长度、AI输出字数限制、最大总结条目数及额外提示词。
8. 直观结果展示：界面分为搜索结果与AI分析两部分，支持高亮关键词与快速文件定位。

## 配置说明

1. 文本源：支持文件输入与直接文本输入两种模式。
2. 关键词设置：输入需搜索的关键字列表，至少包含一个有效词项。
3. 搜索选项：可启用正则模式；设置上下文长度以调整搜索结果的语义范围。
4. AI总结设置：包括期望输出字数、上传条目数量上限、额外提示词等参数。
5. 执行模式：分为“搜索”（初始化阶段）与“AI归纳”（总结阶段）两步，可独立执行或连续运行。

## 原理解析

该工具通过文本分段与关键词索引算法，对文档内容进行多关键字定位与上下文提取。系统在初步检索阶段将结果进行位置排序与交叉合并，生成结构化的文本片段集。在后续分析阶段，工具将这些片段与用户配置参数共同构成提示信息，调用语言模型执行语义归纳生成。整个过程采用异步流式处理与任务队列机制，确保大规模文档分析的高效性与实时响应能力，从而实现从关键词检索到智能总结的完整闭环。

# 错别字检查

错别字检查工具旨在自动识别和修正文本中的拼写错误、标点误用及中英文符号混用问题，帮助用户提高文本的准确性和可读性。它支持对长文本进行分段处理，通过人工智能模型（LLM）分析每段内容，智能生成修正建议，同时保留上下文信息，便于用户理解修改原因和语境，适合文档编辑、内容审核、文本质量提升等场景。

## 功能特性

1. 错别字检测：自动识别文本中的中文和英文拼写错误
2. 标点符号校正：检查中文语段中误用英文标点，或英文语段中误用中文标点
3. 上下文提示：输出错误位置的前后上下文，并给出修正建议和说明
4. AI智能分析：结合用户可选的额外提示词优化检测结果
5. 长文本分段处理：对长文本按句子和长度进行分段，提高检查效果
6. 异步批量处理：支持异步检查大量文本，并在界面实时显示结果

## 配置说明

1. 文本源：指定待检查的文本来源，可从文件或其他输入源读取
2. 最短分段字数：设置每段文本最小长度（默认200字），分段越短，检查越精准，但处理时间更长
3. 额外提示词：可提供附加指令给AI，在满足JSON输出规范的前提下优化检查结果

## 原理解析

该工具通过扫描指定文本源，按配置的最短分段长度拆分文本，然后调用AI模型对每段文本进行检查。AI返回JSON格式的错误列表，每条错误包含上下文、原词、修正词及说明。工具将解析这些结果，并通过将原文分段标记，标识哪些段落存在错误。最终，检查结果以列表形式展示，包括错误原文、修正建议及说明，并提供复制和快速定位功能，确保用户可以直观、快速地理解和处理文本中的错别字与标点问题。处理过程中支持异步操作和实时进度反馈，能够高效处理大批量文本。

# AI服务提供商

AI 服务提供商配置工具用于管理和设置可用的人工智能模型及其连接信息，方便用户选择、添加或删除服务提供商，以便在文本检查、生成或其他 AI 功能中使用。通过该界面，用户可以直观地查看当前已配置的服务列表，并对单个服务提供商的名称、类型、地址、密钥和模型进行详细配置。

## 配置说明

1. 名称：自定义服务提供商名称
2. 类型：选择服务类型，如 Ollama、OpenAI 等。其中，OpenAI表示OpenAI的API，并不仅限于OpenAI公司的AI服务
3. 地址：服务的访问 URL
4. 密钥：用于认证的访问密钥
5. 模型：指定使用的 AI 模型
